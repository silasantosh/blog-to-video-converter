== Description ==

**Blog to Video Converter** instantly transforms your written content into short, social-media-ready videos. It analyzes your post content, images, and data tables to generate a storyboard with zero server-side processing.

== Privacy & Compliance ==

We believe in full transparency. Here is how your data is handled:

1.  **Local Processing**: Video generation is performed entirely within your browser. Your blog content is **not** sent to our servers for video creation.
2.  **No Tracking**: We do not track your site visitors or collect personal data.
3.  **Analytics**: The "Video Analytics" feature runs entirely on your server. Data stays with you.
4.  **External Services**:
    *   **Pexels/Pixabay**: If (and only if) you enter API keys and enable stock media, search keywords are sent to these services to fetch images/videos. This is strictly opt-in.

== Installation ==

1.  Upload the plugin files to the `/wp-content/plugins/blog-to-video-converter` directory.
2.  Activate the plugin through the 'Plugins' screen in WordPress.
3.  Go to **Settings -> Blog to Video** to configure options.
4.  (Optional) Enter Pexels/Pixabay API keys for stock media.

== Frequently Asked Questions ==

= Is this plugin really free? =
Yes, 100% free and GPLv2 licensed. There are no hidden fees.

= Is a donation required? =
No. Donations to the ATG HealthCare Foundation are completely voluntary and do not unlock any special features.

= Where is the video generated? =
In your browser! We use modern web technologies (HTML5 Canvas + Web Audio API) to create the video file on your device.

= Does it work with Custom Post Types? =
Yes! As of version 2.8, it works with all public post types (Products, Events, Portfolios, etc.).
