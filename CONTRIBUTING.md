# Contributing to Blog to Video Converter

Thank you for your interest in contributing to the **Blog to Video Converter**! We welcome contributions of all types.

## ü§ù How to Contribute

### 1. Report Bugs

If you find a bug, please open an issue in the GitHub repository. Provide as much detail as possible, including:

- Your WordPress version
- The browser and OS you are using
- Steps to reproduce the issue

### 2. Suggest Features

Have an idea for a new feature? We'd love to hear it! Open an issue and label it as a "Feature Request".

### 3. Pull Requests

1. Fork the repository.
2. Create a new branch for your feature or bug fix (`git checkout -b feature-new-awesome-thing`).
3. Commit your changes.
4. Push to your branch.
5. Open a Pull Request.

Please be respectful and professional in all interactions.
